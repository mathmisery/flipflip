Flip Flip!

Instructions

Objective: You start with 10 chips. Can you get to 500 chips?

Every time you press "Flip", two coins are flipped and it costs 1 chip. If the number of heads \(N\) equals the number of tails, then you can choose to receive \(N^{2}\) chips in return and reset the game or you can choose to "Continue" play. If the number of heads does not equal the number of tails then you can choose to "Reset" the game or you can choose to "Continue" to play. If you run out of chips, the game is over! "New Game" resets everything and you, well, start a new game. This is a for fun game. Nothing of any monetary value is actually wagered or received. This is to help start a conversation when introducing probability as per this article.
